写给豆豆

发表于 2019-06-07 | 分类于 其他

豆豆，

你来到这个世界已满一年了，不知道你是否习惯，是否喜欢。

眼见着你一点点长大，从一只皱巴巴的外星人变成可爱的小天使，脸蛋儿粉嘟嘟的，像是拉斐尔的画，小嘴巴常常撅成樱桃，眼睛里挂满了小星星。看见你，我就觉得心里暖和，不由自主地笑起来。

我想，你的母亲，也就是我的姐姐，对你会有一些不同看法。因为你常常在夜里把她闹醒，让她无法安睡；常常东抓西扯，让她提心吊胆；常常大喊大叫，让她头晕耳鸣。有了你之后，我的姐姐，迅速地从一个女孩子变成一个女人。你分走了她的一部分生命，不知道她会不会恨你。如果你以后被管得很严，说不定就是她在报仇，小本本上一笔笔都记得很清楚呢。我和她交情不错，你要是以后受欺负，可以找我帮手。放心吧，我会站在你这边的。

你现在还不会说话，和别人交流的时候，除了手舞足蹈，就是咿咿呀呀。相信再过一段日子，你就会奶声奶气地说一些简单的词了。我们这些大人，会说各种各样的话，漂亮话、狠话、坏话、谎话、大话、空话，当然，也有好话、实话、真心话。各种各样的话听得多了，耳朵就装上了过滤器，心也装上了过滤器。你们这些小孩子，随随便便说个词，就能俘获我们的心，看来真是大道至简，大巧若拙。今天是你的生日，照例我要说一些好话，当作给你的祝福。

祝你健康长大。等你长大一些，你就会知道，鲜艳的蘑菇是有毒的。等你再长大一些，你就会知道，不鲜艳的蘑菇，也可能是有毒的。这个世界上有太多毒蘑菇了，让人防不胜防。毒蘑菇会毒害人的身体，也会毒害人的心。如果你不小心吃到毒蘑菇，希望你有足够的好运，吃到解救的药。健康的精神寓于健康的身体，是莫大的幸福。

祝你对世界充满好奇。这个世界上有一个地方叫做监狱，可以有形，也可以无形，有人被迫走进去，也有人主动走进去。住在监狱里的人，双眼无神，面色黯淡，眼前就只剩一方窄窄的土地，人生也再无景色可言。那将是很可怕的。所以我希望你对世界充满好奇，这样你就有了万能钥匙，什么都关不住你了。

祝你勇敢。毫无疑问，这个世界上有无数的大坏蛋，布置好了陷阱，设置好了圈套，等着要捉住你，把你吃掉，非但如此，也许还要逼你吃掉别人。他们手里有刀枪，身上有铠甲。不要怕，要学会保护自己，要学会以牙还牙。即使被吃掉，也要硌掉他们几颗牙。除了和大坏蛋做斗争，还有很多事情都需要勇气，承认自己打碎了水杯需要，对班上小男生表白需要，指出老师的错误需要，走夜路也需要……我想得太远了，还是先等你摆脱学步车，勇敢迈出（literally）人生第一步吧。

祝你有同情心。有人说，同情心是道德的基础。你还不知道什么是同情心，也不知道什么是道德。这些都不重要。重要的是，你看见鸟在天上飞，在林间穿梭，在树上唱歌，会觉得开心；看见鸟在笼子里，在舞台上，在标本室里，会觉得难过。这样就够了。

最后，祝你快乐。

舅舅
